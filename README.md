# toptube

카테고리, 기준, 국가, 기간을 선택하면 조건에 맞는 유튜브 영상 순위를 보여주는 웹사이트.

## 기능
- 카테고리: 전체, 애완동물/동물, 음악, 게임, 뉴스/정치, 인물/블로그, 여행/이벤트, 스포츠, 자동차/교통, 코메디, 엔터테인먼트, 영화/애니메이션, 노하우/스타일, 교육, 과학기술
- 기준: 좋아요 순위, 댓글 순위, 조회수 순위
- 국가: 한국, 미국, 일본, 전세계
- 기간: 일간, 주간, 월간, 연간, 전체

## 개발
```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.
